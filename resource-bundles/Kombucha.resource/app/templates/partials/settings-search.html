The key word(s) and namespece will be used to build appropriate SOQL WHERE clause when querying test classes by their namespeces and names. You could also designate the keyword should appear at start, end, or anywhere in the class name.

For example, SELECT Id, Name FROM ApexClass WHERE Name LIKE 'Test%' AND Name LIKE '%Apex%' AND NamespacePrefix = 'myApp' will return all test classes whose names start with 'Test' and contain 'Apex' while with 'myApp' as their namespaces.
